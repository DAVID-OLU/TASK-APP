
# TASK APP

This application can be used to manage and organize daily activities efficiently. With a friendly user-interface and features like cross-off, delete, and more empowering individuals to accomplish goals and maintain productivity. 


## Deployment

To deploy this project run

```bash
  npm run deploy
```


## Features

- Light/dark mode toggle
- Live previews
- Fullscreen mode
- Cross platform
